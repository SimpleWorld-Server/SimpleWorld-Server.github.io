# 更多附魔

SW为拓展玩法，在原本基础上，引入了10+不影响PVP平衡的生存拓展附魔。

## 获取方式

玩家在附魔台附魔道具时根据类型伴随原生附魔出现，无法在附魔台提前获知，稀有度出现具有最低的等级要求。<br>
附魔种类可使用**/ecoenchants gui** 查看。

## 稀有等级

| 稀有度 | 等级要求 |
|-----|------|
| 普通  | 1级   |
| 不凡  | 8级   |
| 稀有  | 16级  |
| 史诗  | 20级  |
| 传说  | 25级  |
| 稀世  | 30级  |
